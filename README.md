# Powerbi-FarmerMarket-dashboard
This Power BI project showcases my hands-on experience in data analysis, KPI reporting, and dashboard development. It highlights my ability to transform raw data into meaningful insights using Power BI and SQL.

## Tools Used
- Power BI
- SQL
- Excel

## Key KPIs
- Total Quantity
- Revenue by Product
- Date wise Product Trend
- Top Products

## Insights
- Sales increased by 18% 
- The West region generated the highest revenue
- Product category had the highest growth
